#### OSINTを活用した日本周辺のリアルタイム監視アプリ



日本の周辺空域を飛ぶ航空機と、国内の地震情報をリアルタイムで地図上に表示するWebアプリケーションです。



###### 使用技術

バックエンド：Python / Flask

フロントエンド：Leaflet.js



###### 使用API

気象庁 地震情報 API

&#x09;https://www.jma.go.jp/bosai/quake/data/list.json

OpenSky Network API

&#x09;https://opensky-network.org/



アイコン画像

&#x09;https://icons8.jp/



###### 主な機能

・日本周辺の航空機をリアルタイム表示（コールサイン・国籍・高度・速度・針路）

・日本周辺の地震情報の表示、（発生時刻・震源地・最大震度・各地の震度）

・日本周辺の船舶をリアルタイム表示　（船名・MMSI・速度・針路）

・各レイヤーの表示/非表示切り替え

・地震ログ一覧表示からクリックで震源に移動（震度）



###### 工夫した点

OpenSky APIの負荷軽減とリクエスト制限回避のため、サーバー側で1分間キャッシュを実装

航空機マーカーの差分更新により、描画パフォーマンスを大幅に向上

震度に応じた色分けアイコンとポップアップを実装

各地の震度を強い順に並べ、色付けして見やすく表示



使い方・注意事項OpenSky Networkの無料プランでは1日400回のリクエスト制限があります。

アプリを快適にご利用いただくため、OpenSkyへのユーザー登録を強く推奨します。登録後、以下の環境変数を設定してください：env



OPENSKY\_USER=あなたのユーザー名

OPENSKY\_PASS=あなたのパスワード

AISSTREAM\_KEY=あなたのAISSTREAMAPIKEY


※必ずしも正しい場所に表示されているわけではありません。留意してください。
